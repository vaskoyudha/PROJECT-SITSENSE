IoT Showcase Section

Section yang menjelaskan secara detail komponen IoT yang digunakan dalam SitSense, mulai dari sensor hingga cloud gateway.

1

Sensor Tekanan Kursi

Panel sensor piezo-elektrik mendeteksi distribusi berat tubuh dan mengirimkan data matrix tekanan. Ideal untuk memantau postur duduk menit demi menit.

2

NodeMCU / ESP32 Gateway

Microcontroller ESP32 menggabungkan Wi‑Fi, BLE, dan ADC sehingga data sensor langsung ditransmisikan ke cloud SitSense, lengkap dengan enkripsi TLS.

3

Gateway Cloud & AI Engine

Data yang masuk dianalisis oleh machine learning posture model untuk menghasilkan skor postur, rekomendasi Gemini AI, dan pengingat adaptif.

4

Dashboard & Alert Layer

UI SitSense merender data secara realtime, menyalakan notifikasi, dan menampilkan pembaruan streak agar kebiasaan duduk sehat terjaga.
